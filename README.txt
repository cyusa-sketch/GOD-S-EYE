# God's Eye Company – Website
Static site using Tailwind, Animate.css, Font Awesome, and GSAP.

## How to run
Open `index.html` in a browser, or deploy the folder on Vercel/Netlify.

## Replace images
All section images are under `assets/`:
- hero.jpg (Smart Call Center flyer) – used in hero, services card, footer logo.
- benefits.jpg – used in Benefits section.
- mission-vision.jpg – used in About section.
- industries.jpg – used in Industries section.
